# Áudio de Chamada de Motoristas

Para adicionar um áudio personalizado para a chamada de motoristas:

1. Coloque um arquivo de áudio chamado `chamada_motorista.mp3` nesta pasta
2. O áudio será reproduzido automaticamente quando um motorista for chamado
3. Se o arquivo não existir, o sistema usará a Web Speech API do navegador como fallback

**Formato recomendado:**
- Nome do arquivo: `chamada_motorista.mp3`
- Formato: MP3
- Duração: 3-5 segundos (será repetido automaticamente)

**Exemplo de mensagem de áudio:**
"Atenção! Chamada para o motorista [NOME]. Compareça à portaria."
